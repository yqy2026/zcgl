# 合同模块 V2 升级 - 手动验证步骤

请按照以下步骤手动验证合同模块 V2 的新增功能。

**系统地址**: [http://192.168.1.90:5173/](http://192.168.1.90:5173/)

## 1. 验证 V2 新增字段 (创建合同)

1.  **进入页面**: 导航至「租赁管理」 -> 「合同管理」。
2.  **点击新增**: 点击列表右上角的「新增合同」按钮。
3.  **验证关联资产 (多选)**:
    -   在「关联信息」区域，点击「关联资产」下拉框。
    -   尝试选择多个资产（例如：选择 Asset A，再选择 Asset B）。
    -   **预期结果**: 能够同时选中多个资产。
4.  **验证合同类型 & 服务费率**:
    -   在「基本信息」区域，找到「合同类型」。
    -   选择「委托运营协议」。
    -   **预期结果**: 「付款周期」下方应动态出现「服务费率」输入框。
    -   输入 `0.05` (表示 5%)。
    -   切换回「租赁合同（下游）」。
    -   **预期结果**: 「服务费率」输入框应消失。
5.  **验证付款周期**:
    -   点击「付款周期」下拉框。
    -   **预期结果**: 包含选项：月付、季付、半年付、年付。
6.  **验证经营用途**:
    -   在「承租方信息」区域，找到「经营用途」。
    -   **预期结果**: 存在该文本域输入框。

## 2. 验证合同列表展示

1.  **保存合同**: 填写完必要信息后，保存合同。
2.  **查看列表**: 返回合同列表页。
3.  **验证多资产展示**:
    -   找到刚才创建的合同。
    -   **预期结果**: 「物业名称」列若关联多个资产，应显示如 `Asset A, Asset B` 或通过标签/Tooltip 展示。

## 3. 验证续签功能

1.  **选择合同**: 在列表中找到状态为「有效」的合同。
2.  **点击续签**: 点击操作列的「续签」按钮。
3.  **验证表单**:
    -   **预期结果**: 跳转到新建/续签页面，且原合同信息（承租方、资产等）已自动带入。
    -   **预期结果**: 有标记表明这是基于原合同 `[合同号]` 的续签。

## 4. 验证终止功能

1.  **选择合同**: 在列表中找到状态为「有效」的合同。
2.  **点击终止**: 点击操作列的「终止」按钮。
3.  **验证弹窗**:
    -   **预期结果**: 弹出终止确认框。
    -   输入「终止日期」（默认为今天）。
    -   输入「终止原因」。
    -   确认是否退还押金。
4.  **执行终止**: 点击确定。
5.  **验证状态**:
    -   列表页刷新后，该合同状态应变为「已终止」。

---
**注意**: 如果页面加载失败，请确认后端服务 (`port 8002`) 和前端服务 (`port 5173`) 均已正常启动。
