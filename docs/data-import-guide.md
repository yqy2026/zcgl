# 数据导入指南

## 📋 Excel数据导入完整教程

本指南将详细介绍如何使用Excel批量导入资产数据。

## 准备工作

### 1. 下载导入模板
1. 登录系统，进入"数据管理" → "数据导入"
2. 点击"下载导入模板"按钮
3. 保存模板文件到本地

### 2. 了解模板结构
模板包含以下必填字段：
- **物业名称** - 资产的名称标识
- **权属方** - 资产的法定所有者
- **所在地址** - 资产的详细地址
- **确权状态** - 已确权/未确权/部分确权
- **物业性质** - 经营类/非经营类
- **使用状态** - 出租/闲置/自用/公房/其他

可选字段：
- 经营管理方、各类面积、合同信息等

## 数据准备

### 1. 数据格式要求
```excel
物业名称    | 权属方      | 所在地址        | 确权状态 | 物业性质 | 使用状态
测试大厦A座 | ABC公司     | 北京市朝阳区... | 已确权   | 经营类   | 出租
测试大厦B座 | ABC公司     | 北京市朝阳区... | 已确权   | 经营类   | 闲置
```

### 2. 数据验证规则
- **物业名称**：不能为空，建议唯一
- **权属方**：不能为空，长度不超过200字符
- **地址**：不能为空，长度不超过500字符
- **面积字段**：必须为数字，不能为负数
- **日期字段**：格式为YYYY-MM-DD或YYYY年MM月DD日

### 3. 常见数据问题
❌ **错误示例**：
```
物业名称: (空白)           → 必填字段不能为空
面积: -100                → 面积不能为负数
日期: 2023/13/45          → 日期格式错误
确权状态: 已经确权         → 不在允许值范围内
```

✅ **正确示例**：
```
物业名称: 中央商务大厦A座
面积: 1500.5
日期: 2023-12-01
确权状态: 已确权
```

## 导入操作

### 1. 上传文件
1. 点击"选择文件"或拖拽文件到上传区域
2. 支持的格式：.xlsx、.xls
3. 文件大小限制：10MB
4. 点击"上传"按钮

### 2. 数据预览
- 系统会显示前10行数据供预览
- 检查数据是否正确读取
- 确认字段映射是否正确

### 3. 验证结果
系统会自动验证数据：
- ✅ **验证通过**：显示绿色对勾
- ❌ **验证失败**：显示红色错误信息
- ⚠️ **警告**：显示黄色警告信息

### 4. 处理错误
如果有验证错误：
1. 点击"下载错误报告"
2. 根据错误信息修正Excel文件
3. 重新上传修正后的文件

### 5. 确认导入
- 检查导入摘要信息
- 选择重复数据处理方式：
  - 跳过重复数据
  - 更新重复数据
  - 全部导入
- 点击"确认导入"

## 高级功能

### 1. 字段映射
如果您的Excel列名与模板不同：
1. 在预览页面点击"字段映射"
2. 手动指定每列对应的系统字段
3. 保存映射配置供下次使用

### 2. 数据转换
系统支持自动数据转换：
- **日期格式**：自动识别多种日期格式
- **数字格式**：自动去除千分位分隔符
- **文本清理**：自动去除前后空格

### 3. 批量更新
导入时可以选择更新策略：
- **仅新增**：只导入新数据，跳过已存在的
- **覆盖更新**：用新数据覆盖已存在的记录
- **智能合并**：只更新有变化的字段

## 最佳实践

### 1. 数据准备建议
- 📊 **分批导入**：建议每次导入不超过1000条记录
- 🔍 **数据清理**：导入前检查数据完整性和准确性
- 💾 **备份原始数据**：保留原始Excel文件作为备份
- 🏷️ **统一命名**：保持权属方、地址等字段的命名一致性

### 2. 导入流程建议
```
准备数据 → 小批量测试 → 验证结果 → 批量导入 → 数据检查
```

### 3. 错误处理策略
1. **预处理**：使用Excel的数据验证功能预先检查
2. **分步导入**：先导入基本信息，再补充详细信息
3. **增量导入**：定期导入新增和变更的数据

## 常见问题解答

### Q: 导入时提示"文件格式不支持"？
A: 请确保文件格式为.xlsx或.xls，不支持.csv格式的直接导入。

### Q: 为什么有些数据导入失败？
A: 常见原因：
- 必填字段为空
- 数据格式不正确
- 字符长度超限
- 枚举值不在允许范围内

### Q: 如何处理重复数据？
A: 系统根据"物业名称+权属方"判断重复，您可以选择：
- 跳过重复记录
- 更新重复记录
- 重命名后导入

### Q: 导入后发现数据错误怎么办？
A: 可以通过以下方式修正：
- 在资产列表中逐个编辑
- 修正Excel文件后重新导入覆盖
- 使用批量编辑功能

### Q: 导入进度很慢怎么办？
A: 建议：
- 减少单次导入的数据量
- 检查网络连接状态
- 避免在系统繁忙时段导入
