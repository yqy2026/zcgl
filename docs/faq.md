## 登录和账户问题

### Q1: 忘记密码怎么办？
**A:** 
1. 在登录页面点击"忘记密码"链接
2. 输入您的用户名或邮箱地址
3. 系统会发送重置密码的邮件到您的邮箱
4. 点击邮件中的链接设置新密码
5. 如果没有收到邮件，请检查垃圾邮件文件夹或联系管理员

### Q2: 登录时提示"用户名或密码错误"？
**A:** 
- 检查用户名和密码是否输入正确
- 注意大小写敏感
- 确认Caps Lock键是否开启
- 如果多次尝试失败，账户可能被临时锁定，请等待15分钟后重试

### Q3: 登录后页面显示空白？
**A:** 
1. 刷新浏览器页面（F5或Ctrl+R）
2. 清除浏览器缓存和Cookie
3. 尝试使用无痕/隐私模式访问
4. 检查浏览器是否支持（推荐Chrome、Firefox、Safari、Edge最新版本）

### Q4: 如何修改个人信息？
**A:** 
1. 点击右上角用户头像
2. 选择"个人资料"
3. 修改相关信息后点击"保存"

## 资产管理问题

### Q5: 如何快速查找特定资产？
**A:** 
- **快速搜索**：在页面顶部搜索框输入物业名称、权属方或地址关键词
- **高级筛选**：点击"高级搜索"，使用多个条件组合筛选
- **保存筛选**：将常用的筛选条件保存为快捷筛选
- **排序功能**：点击列标题进行排序

### Q6: 出租率是如何计算的？
**A:** 
- **自动计算**：出租率 = (已出租面积 ÷ 可出租面积) × 100%
- **手动输入**：也可以直接在出租率字段输入数值
- **实时更新**：当修改已出租面积或可出租面积时，出租率会自动重新计算
- **显示规则**：只有经营类物业才显示出租率

### Q7: 如何批量修改资产信息？
**A:** 
1. 在资产列表中勾选需要修改的资产
2. 点击"批量操作"按钮
3. 选择"批量编辑"
4. 在弹出的表单中填写要修改的字段（空白字段不会被修改）
5. 点击"确认修改"

### Q8: 删除资产后能恢复吗？
**A:** 
- 删除的资产会保留在变更历史中
- 系统管理员可以从数据库备份中恢复
- 建议删除前先导出数据作为备份
- 重要资产建议使用"停用"而不是删除

### Q9: 如何查看资产的历史变更记录？
**A:** 
1. 进入资产详情页面
2. 点击"变更历史"标签
3. 可以查看所有的修改记录，包括：
   - 修改时间和操作人
   - 修改的字段
   - 修改前后的值对比

## 数据导入导出问题

### Q10: Excel导入时出现"格式错误"？
**A:** 
- 确保使用系统提供的导入模板
- 检查必填字段是否都已填写
- 确认数据格式是否正确（日期、数字等）
- 查看详细错误报告，逐行修正问题
- 文件格式必须是.xlsx或.xls

### Q11: 导入的数据部分成功，部分失败？
**A:** 
- 这是正常现象，系统会导入验证通过的数据
- 下载错误报告查看失败原因
- 修正失败的数据后可以再次导入
- 系统会自动跳过已存在的重复数据

### Q12: 如何导出特定条件的数据？
**A:** 
1. 使用筛选功能设置导出条件
2. 点击"导出"按钮
3. 选择"导出筛选结果"
4. 选择导出格式（Excel或CSV）
5. 选择要导出的字段

### Q13: 导出的Excel文件打开乱码？
**A:** 
- 使用较新版本的Excel打开
- 如果仍有乱码，尝试用记事本打开CSV文件，另存为UTF-8编码
- 或者在Excel中使用"数据" → "从文本"功能导入

## 系统性能问题

### Q14: 系统运行缓慢怎么办？
**A:** 
1. **检查网络**：确保网络连接稳定
2. **清理缓存**：清除浏览器缓存和Cookie
3. **关闭标签页**：减少同时打开的浏览器标签页
4. **重启浏览器**：完全关闭浏览器后重新打开
5. **检查系统资源**：确保电脑内存和CPU使用率不过高

### Q15: 页面加载时间很长？
**A:** 
- 检查网络连接速度
- 尝试在网络较好的时段使用
- 使用有线网络而不是WiFi
- 联系网络管理员检查网络配置

### Q16: 大量数据操作时系统卡顿？
**A:** 
- 分批处理大量数据，避免一次性操作过多记录
- 使用筛选功能减少显示的数据量
- 在系统使用较少的时段进行大批量操作
- 考虑使用导入导出功能处理大量数据

## 技术支持问题

### Q17: 支持哪些浏览器？
**A:** 
推荐使用以下浏览器的最新版本：
- ✅ Google Chrome 90+
- ✅ Mozilla Firefox 88+
- ✅ Safari 14+
- ✅ Microsoft Edge 90+
- ❌ 不支持Internet Explorer

### Q18: 移动设备能使用吗？
**A:** 
- 系统支持响应式设计，可以在手机和平板上使用
- 推荐使用平板电脑以获得更好的体验
- 复杂操作建议在电脑上完成
- 移动端主要用于查看和简单编辑

### Q19: 数据安全性如何保障？
**A:** 
- 所有数据传输使用HTTPS加密
- 定期自动备份数据
- 用户权限严格控制
- 操作日志完整记录
- 服务器采用专业安全防护

### Q20: 系统维护时间是什么时候？
**A:** 
- **定期维护**：每周日凌晨2:00-4:00
- **紧急维护**：会提前通知用户
- **维护通知**：通过邮件和系统公告发布
- **数据备份**：维护前会自动备份所有数据

- **E004**：文件上传失败 → 检查文件大小和格式
- **E005**：服务器内部错误 → 联系技术支持
