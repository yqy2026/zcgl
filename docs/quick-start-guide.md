# 快速入门指南

## 🚀 5分钟快速上手

欢迎使用土地房产资产管理系统！本指南将帮助您在5分钟内掌握系统的基本操作。

### 第一步：登录系统
1. 打开浏览器，访问系统地址
2. 输入您的用户名和密码
3. 点击"登录"按钮

### 第二步：熟悉界面
```
┌─────────────────────────────────────────────────────────┐
│  🏠 土地房产资产管理系统    🔔 通知  👤 用户菜单        │
├─────────────┬───────────────────────────────────────────┤
│ 📊 数据看板  │                                           │
│ 🏢 资产管理  │              主要工作区域                  │
│ 📋 数据管理  │                                           │
│ 📈 数据分析  │                                           │
│ ⚙️ 系统管理  │                                           │
└─────────────┴───────────────────────────────────────────┘
```

### 第三步：查看数据看板
- 点击左侧菜单"数据看板"
- 查看关键指标：资产总数、总面积、出租率等
- 浏览可视化图表了解资产分布情况

### 第四步：浏览资产列表
- 点击"资产管理" → "资产列表"
- 切换表格视图和卡片视图
- 尝试搜索功能：在搜索框输入关键词

### 第五步：新增第一个资产
1. 点击"新增资产"按钮
2. 填写基本信息：
   - 物业名称：测试物业A
   - 权属方：测试公司
   - 地址：测试地址123号
3. 选择状态信息：
   - 确权状态：已确权
   - 物业性质：经营类
   - 使用状态：出租
4. 点击"保存"

### 🎉 恭喜！您已完成基本操作
