# 合同PDF智能导入字段提取问题分析报告

## 📊 总体情况

基于全面测试，当前字段提取成功率约为 **94.7%** (18/19字段)，但仍存在一些需要改进的问题。

## 🔍 主要问题分析

### 1. 合同编号提取问题

**问题描述**:
- ❌ "包装合字（2025）第022号" → 未提取
- ❌ "【2025】租字第028号" → 未提取
- ❌ "租赁合同第001号" → 未提取
- ✅ "合同编号：ZC-2025-022" → 正确提取
- ✅ "编号：20250315-001" → 正确提取

**问题原因**:
- 中文字符和特殊符号（如括号、【】）的匹配模式不够全面
- 缺少针对中文合同编号格式的专门模式

**改进建议**:
```python
# 增加中文合同编号模式
patterns = [
    r'[【\[]([^】\]]+)[】\]]第?\s*\d+[号]',
    r'([^（）\s]+[（\[][）\]]第\s*\d+[号]',
    r'合同[编号号]([^\n\r]{3,20})',
    r'包装合字[^第]*第(\d+)[号]',
    r'租字[第]*(\d+)[号]'
]
```

### 2. 承租方/出租方识别问题

**问题描述**:
- 承租方和出租方信息提取不准确，有时混淆
- 多个关键词提取时混合了多个字段的内容

**问题原因**:
- 角色识别逻辑不够精确
- 缺少上下文理解能力
- 正则表达式匹配范围过大

**改进建议**:
```python
# 改进角色匹配模式
"tenant_name": {
    "keywords": ["承租方", "租客", "乙方", "承租人", "租用方"],
    "patterns": [
        r'承租方[：:\s]*([^\n\r，,。.]{2,50})',
        r'租客[：:\s]*([^\n\r，,。.]{2,50})',
        r'乙方[：:\s]*([^\n\r，,。.]{2,50})(?=甲方|支付|租金)'
    ]
},
"landlord_name": {
    "keywords": ["出租方", "房东", "甲方", "出租人", "物业方"],
    "patterns": [
        r'出租方[：:\s]*([^\n\r，,。.]{2,50})',
        r'房东[：:\s]*([^\n\r，,。.]{2,50})',
        r'甲方[：:\s]*([^\n\r，,。.]{2,50})(?=乙方|承租|租金)'
    ]
}
```

### 3. 地址提取问题

**问题描述**:
- 地址信息被截断或提取不完整
- 多行地址无法正确合并

**问题原因**:
- 单行匹配模式限制了地址长度
- 缺少多行地址的合并逻辑

**改进建议**:
```python
# 改进地址提取模式
patterns = [
    r'(?:物业地址|租赁物业|地址|房屋地址)[：:\s]*([^\n\r]{10,100})',
    r'(?:物业名称|楼盘名称)[：:\s]*([^\n\r]{5,50})',
    # 多行地址合并
    r'([^\n\r]{5,}(?:市|区|县|镇|街道|路|大道|街|巷|号|栋|座|层|室|单元)[^\n\r]{0,50})'
]
```

### 4. 支付条款提取问题

**问题描述**:
- 支付方式能提取，但支付条款提取不完整
- "按月支付"等周期信息丢失

**问题原因**:
- 支付条款的匹配模式过于简单
- 缺少周期性支付模式的识别

**改进建议**:
```python
"payment_terms": {
    "keywords": ["支付", "付款", "缴费", "周期", "频率"],
    "patterns": [
        r'(?:按|每)(月|季|半年|年)支付',
        r'(?:月付|季付|半年付|年付)',
        r'支付周期[：:\s]*([^\n\r，,。.]{2,20})',
        r'(?:每月|每季度|每半年|每年).*?支付'
    ]
}
```

### 5. 物业名称提取问题

**问题描述**:
- 物业名称被其他字段信息干扰
- 提取结果包含了额外的信息

**改进建议**:
```python
"property_name": {
    "keywords": ["物业名称", "楼盘名称", "房产名称", "建筑名称"],
    "patterns": [
        r'物业名称[：:\s]*([^\n\r，,。.]{2,30})(?=\s|$|[，,。.])',
        r'楼盘名称[：:\s]*([^\n\r，,。.]{2,30})(?=\s|$|[，,。.])',
        r'([^，,。.\n\r]*(?:大厦|广场|中心|大厦|商业楼|写字楼)[^\n\r，,。.]{0,10})'
    ]
}
```

### 6. 实用面积提取问题

**问题描述**:
- 实用面积(rented_area)完全无法提取
- 面积字段的优先级和区分逻辑需要改进

**改进建议**:
```python
"rented_area": {
    "keywords": ["实用面积", "使用面积", "净面积", "套内面积"],
    "patterns": [
        r'(?:实用|使用|净|套内)面积[：:\s]*([0-9,，.]+)\s*平方米?',
        r'实用面积[：:\s]*([0-9,，.]+)\s*[㎡㎡]'
    ]
}
```
