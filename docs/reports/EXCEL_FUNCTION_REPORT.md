## 📊 技术实现

### 使用的技术栈
- **后端框架**: FastAPI
- **Excel处理**: pandas + openpyxl
- **文件上传**: FastAPI UploadFile
- **数据验证**: Pydantic schemas

### 支持的Excel字段
1. 物业名称 (必填)
2. 地址 (必填)
3. 确权状态 (必填)
4. 物业性质 (必填)
5. 使用状态 (必填)
6. 权属方 (必填)
7. 经营管理方 (可选)
8. 业态类别 (可选)
9. 总面积 (可选)
10. 可使用面积 (可选)
11. 是否涉诉 (默认"否")
12. 备注 (可选)

## 🚀 使用示例

### 1. 下载模板
```bash
curl -O http://localhost:8001/api/v1/excel/template
```

### 2. 导出数据
```bash
curl "http://localhost:8001/api/v1/excel/export?property_nature=经营性" -O
```

### 3. 导入数据
```bash
curl -X POST "http://localhost:8001/api/v1/excel/import" \
  -F "file=@assets.xlsx" \
  -F "skip_errors=true"
```
