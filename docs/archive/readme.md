# 归档目录说明

`docs/archive/` 用于存放历史草稿、阶段性复盘、过期计划与不再作为当前基线的文档。

## 规则

- 归档文档不作为当前需求或实现基线。
- 新增归档时无需修改此文件，按子目录归类存放即可。
- 引用归档内容时应同时给出现行基线文档链接。

## 子目录说明

| 子目录 | 内容 |
|--------|------|
| `frontend-reports/` | 前端 AI 任务执行报告（2026-02，已归档） |
| `backend-plans/` | 后端已完结技术方案（从 `plans/` 归档） |
| `terminology/` | 词汇表审计记录（已冻结） |
