# 需求决策归档

本目录用于归档需求形成过程中的历史材料。

可放入本目录的内容：

- 访谈冻结结论。
- 评审意见与采纳过程。
- As-Built 字段对照。
- 旧字段下线策略。
- 历史迁移说明。
- 已被目标态文档吸收的临时决策记录。

不应放入本目录的内容：

- 当前产品目标态。
- 当前字段和 API 契约。
- 当前实现追踪矩阵。

当前产品目标态见 `docs/prd.md`。
