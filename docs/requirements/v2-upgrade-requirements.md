## 一、版本概述

### 1.1 V1.0 现状

土地物业资产管理系统 V1.0 已实现：
- ✅ 用户认证与权限管理（RBAC）
- ✅ 资产基础信息管理
- ✅ 租赁合同基础管理
- ✅ 租金台账基础功能
- ✅ 权属方/项目列表管理
- ✅ 数据字典管理

### 1.2 V2.0 升级目标

V2.0 版本将重点解决以下问题：

| 问题 | V1.0 现状 | V2.0 目标 |
|------|----------|----------|
| 业务模型 | 两方关系（权属方↔租户） | 三方关系（权属方↔运营方↔租户） |
| 合同管理 | 仅列表页 | 完整详情页+编辑页+续签流程 |
| 权属方管理 | 仅列表页 | 完整详情页+编辑页+收支统计 |
| 项目管理 | 仅列表页 | 完整详情页+资产归集视图 |
| 审批流程 | 无 | 可配置审批引擎 |
| 消息通知 | 无 | 站内消息+企业微信推送 |
| 财务模块 | 基础台账 | 完整收款管理+欠款统计+催缴 |

### 1.3 核心业务模型

V2.0 系统支持**三方业务关系**：

```
权属方 ←──[上游合同]── 运营方 ←──[下游合同]── 终端租户
```

#### 两种合作模式

| 模式 | 资金流向 | 运营方收益来源 |
|------|----------|----------------|
| **承租方式** | 终端租户 → 运营方 → 权属方 | 租金差价 |
| **委托运营** | 终端租户 → 权属方 → 运营方 | 服务费（百分比） |

#### 特殊场景

- 运营方和权属方可能是**同一实体**
- 同一权属方下的资产可以**同时存在两种模式**
- 后续需支持**多个运营方**

## 二、功能模块需求

### 2.1 合同管理模块 ⭐ 核心模块

#### 2.1.1 合同类型

| 类型 | 描述 | 当事方 |
|------|------|--------|
| **上游合同** | 与权属方签订 | 运营方（乙方） ↔ 权属方（甲方） |
| **下游合同** | 与终端租户签订 | 运营方（甲方） ↔ 终端租户（乙方） |

上游合同分为：
- 承租合同（运营方承租资产）
- 委托运营协议（权属方委托运营）

#### 2.1.2 合同核心字段

**合同本体信息**：

| 字段 | 说明 | 备注 |
|------|------|------|
| 合同编号 | 唯一标识 | 系统自动生成 |
| 合同类型 | 承租/委托运营/租赁 | 必填 |
| 甲方 | 合同甲方 | 必填 |
| 乙方 | 合同乙方 | 必填 |
| 签订日期 | 合同签署日期 | 必填 |
| 生效日期 | 合同开始生效 | 必填 |
| 到期日期 | 合同截止日期 | 必填 |
| 付款周期 | 月付/季付/年付 | 必填 |
| 租赁保证金 | 押金金额（固定值） | 必填 |
| 合同状态 | 草稿/正常/已到期/已终止/已续签 | 系统管理 |
| 合同附件 | PDF扫描件 | 支持多个 |

**下游合同额外字段**：

| 字段 | 说明 |
|------|------|
| 租户联系人 | 租户方联系人姓名 |
| 用途说明 | 租赁用途描述 |

#### 2.1.3 租金条款

租金条款支持**阶梯递增**，一份合同包含多个租金条款：

| 字段 | 说明 |
|------|------|
| 开始日期 | 条款生效开始 |
| 结束日期 | 条款生效结束 |
| 月租金 | 该时段的月租金额 |

**示例**（3年合同，年递增5%）：

| 时间段 | 月租金 |
|--------|--------|
| 2024/01 - 2024/12 | 10,000元 |
| 2025/01 - 2025/12 | 10,500元 |
| 2026/01 - 2026/12 | 11,025元 |

#### 2.1.4 合同与资产关系

- **多对多关系**：一份合同可关联多个资产，一个资产可被多份合同覆盖
- **唯一有效租金规则**：同一资产在任一时间点，只允许一套有效租金条款
- **冲突处理**：录入新合同时如有时间段冲突，提示用户选择是否覆盖

#### 2.1.5 合同功能清单

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 创建合同 | 新建上游/下游合同 | P0 |
| 查看合同详情 | 展示合同完整信息+租金条款+关联资产 | P0 |
| 编辑合同 | 修改合同信息（需审批控制） | P0 |
| 删除合同 | 删除合同（需审批控制） | P0 |
| 合同附件 | 上传/查看PDF扫描件 | P0 |
| 合同续签 | 预填数据快捷创建+自动结束原合同+转移押金 | P0 |
| 合同终止 | 提前终止+退租结算 | P0 |
| 合同变更记录 | 记录所有修改历史 | P0 |
| 合同到期提醒 | 到期前N天自动提醒 | P0 |
| 合同审批 | 可配置的审批流程 | P1 |

| 功能 | 描述 | 优先级 |

| 录入实收 | 手动录入实际收款金额、日期、方式 | P0 |
| 欠款计算 | 自动计算欠款（应收-实收） | P0 |
| 逾期统计 | 计算逾期天数 | P0 |
| 汇总统计 | 按项目/权属方/租户维度汇总 | P0 |
| 催缴管理 | 逾期提醒、催缴记录 | P1 |

## 四、非功能需求

### 4.1 移动端

- **当前阶段**：PC端为主
- **后续规划**：独立APP 或 企业微信应用

### 4.2 性能要求

- 列表页支持大数据量分页（1000+记录）
- 报表生成响应时间 < 5秒

### 4.3 安全要求

- 基于角色的权限控制（RBAC）
- 操作日志审计
- 数据变更历史追踪
