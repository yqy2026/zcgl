# 资产管理 PRD

## ✅ Status
**当前状态**: Draft (2026-02-03)

## 背景与目标
统一资产信息管理流程，提升数据准确性与可检索性。

## 目标范围
- 资产档案管理（字段维护、查询、导入/导出）
- 资产与项目/权属方/产权证关联
- 资产统计与分析基础能力

## 非目标
- 资产估值模型
- 高级 BI 报表体系

## 关键需求
- 资产字段完整、可扩展
- 高效筛选与批量操作
- 数据完整性校验

## 验收标准
- 资产 CRUD 可用
- 关键筛选与导出功能可用
- 关联关系正确保存
