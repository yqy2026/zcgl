# AssetDetailPage 组件文档

## 概述

AssetDetailPage 是资产详情页面组件，用于展示单个资产的详细信息和变更历史。

## 功能特性

### 核心功能
- ✅ 资产详细信息展示
- ✅ 变更历史记录查看
- ✅ 资产编辑操作
- ✅ 资产删除操作
- ✅ 资产信息复制
- ✅ 页面打印功能

### 用户体验
- ✅ 响应式布局设计
- ✅ 加载状态显示
- ✅ 错误处理和边界保护
- ✅ 友好的用户反馈
- ✅ 打印样式优化

## 组件结构

```
AssetDetailPage/
├── AssetDetailPage.tsx          # 主组件
├── AssetDetailPage.module.css   # 样式文件
└── AssetDetailPage.md          # 文档文件
```

## 使用方式

### 路由配置
```typescript
import AssetDetailPage from '@/pages/AssetDetailPage'

// 在路由配置中使用
{
  path: '/assets/:id',
  element: <AssetDetailPage />
}
```

### URL 参数
- `id`: 资产ID（必需）

## 依赖组件

### 子组件
- `AssetDetailInfo`: 资产详细信息展示组件
- `AssetHistory`: 资产变更历史组件
- `ErrorBoundary`: 错误边界组件

### 服务依赖
- `assetService`: 资产相关API服务
- `useQuery`: 数据获取Hook
- `useMutation`: 数据变更Hook

## 状态管理

### 本地状态
- `activeTab`: 当前激活的标签页
- `asset`: 资产详情数据
- `isLoading`: 加载状态
- `error`: 错误信息

### 查询状态
- `['asset', id]`: 资产详情查询
- `deleteMutation`: 删除资产变更

## 用户操作

### 主要操作
1. **返回列表**: 导航回资产列表页面
2. **编辑资产**: 跳转到资产编辑页面
3. **删除资产**: 删除当前资产（需确认）
4. **复制信息**: 复制资产基本信息到剪贴板
5. **打印页面**: 打印当前页面内容

### 标签页切换
- **基本信息**: 展示资产的详细信息
- **变更历史**: 展示资产的变更记录

## 错误处理

### 加载错误
- 网络错误时显示重试选项
- 资产不存在时显示友好提示
- 权限错误时引导用户操作

### 操作错误
- 删除失败时显示错误消息
- 复制失败时提供备选方案
- 网络异常时自动重试
