- **错误分类**: 按严重程度分类（低、中、高、严重）
- **重试机制**: 智能重试，支持最大重试次数限制
- **错误历史**: 维护错误历史，支持查看详细信息
- **自动隐藏**: 可配置的自动隐藏机制（5秒）

```typescript
// 使用示例

  type: 'error',

```

- **时间估算**: 智能的剩余时间估算
- **步骤分解**: 支持子步骤和复杂任务分解

```typescript

```

#### 特性亮点

## 使用指南

### 集成方式
```typescript
// 1. 在应用顶层集成
import {
  SmartLoadingProvider,
  SmartErrorHandler,
  SmartProgressProvider,
  SmartInteractionProvider,
  SmartResponseProvider
} from './components/UX'

function App() {
  return (
    <SmartLoadingProvider>
      <SmartErrorHandler>
        <SmartProgressProvider>
          <SmartInteractionProvider>
            <SmartResponseProvider>
              <YourAppComponents />
            </SmartResponseProvider>
          </SmartInteractionProvider>
        </SmartProgressProvider>
      </SmartErrorHandler>
    </SmartLoadingProvider>
  )
}

// 2. 在特定页面使用
import { useSmartLoading } from './components/UX'

const Component = () => {
  const { showLoading, hideLoading } = useSmartLoading()

  const handleUpload = async () => {
    showLoading({ type: 'upload', message: '上传中...' })

    try {
      // 上传逻辑
      await uploadFile()
      hideLoading()
      showNotification({ type: 'success', message: '上传成功' })
    } catch (error) {
      hideLoading()
      showError({ type: 'error', message: error.message })
    }
  }

  return <YourComponent onUpload={handleUpload} />
}
```
