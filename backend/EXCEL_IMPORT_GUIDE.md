# 土地物业资产管理系统 - Excel导入模板使用指南

## 模板说明

本模板已优化与新增资产表单字段保持一致，支持完整的资产信息导入。

## 必填字段（*）

### 基本信息
- **权属方*** - 资产权属单位名称
- **物业名称*** - 物业具体名称
- **所在地址*** - 物业详细地址
- **权属类别** - 权属分类（如：国有、集体、私有等）
- **项目名称** - 所属项目名称

### 面积信息
- **实际房产面积*** - 实际房产面积（平方米）
- **土地面积** - 土地面积（平方米）
- **可出租面积** - 可用于出租的面积（平方米）
- **已出租面积** - 已出租的面积（平方米）
- **未出租面积** - 未出租的面积（平方米）
- **非经营物业面积** - 非经营性物业面积（平方米）

### 状态信息
- **确权状态*** - 已确权、未确权、部分确权
- **物业性质*** - 经营类、非经营类
- **使用状态*** - 出租、闲置、自用、公房、其他
- **证载用途** - 产权证载明的用途
- **实际用途** - 实际使用用途
- **业态类别** - 具体业态分类
- **是否涉诉** - 是否涉及诉讼（是/否）

### 接收信息
- **接收模式** - 资产接收模式
- **接收协议开始日期** - 接收协议开始日期（YYYY-MM-DD）
- **接收协议结束日期** - 接收协议结束日期（YYYY-MM-DD）
- **接收协议文件** - 接收协议文件名列表（用逗号分隔）

### 合同信息
- **租赁合同编号** - 租赁合同编号
- **合同开始日期** - 合同开始日期（YYYY-MM-DD）
- **合同结束日期** - 合同结束日期（YYYY-MM-DD）
- **租户名称** - 租户名称

### 终端合同信息
- **终端合同文件** - 终端合同文件名列表（用逗号分隔）

## 字段填写规范

### 枚举字段规范
- **确权状态**: 已确权、未确权、部分确权
- **物业性质**: 经营类、非经营类
- **使用状态**: 出租、闲置、自用、公房、其他
- **是否涉诉**: 是、否

### 日期字段规范
- 格式: YYYY-MM-DD (如: 2024-01-15)
- 支持格式: 2024年1月15日、2024/01/15、2024-01-15

### 数值字段规范
- 面积单位: 平方米
- 数值类型: 数字，支持小数点后两位
- 负数: 不支持负数

### 文件字段规范
- 多文件: 用英文逗号分隔
- 示例: file1.pdf,file2.pdf,file3.pdf

## 自动计算字段

以下字段会自动计算，无需填写：
- **出租率** = 已出租面积 ÷ 可出租面积 × 100%
- **未出租面积** = 可出租面积 - 已出租面积

## 向后兼容

模板支持以下旧字段名称（自动映射）：
- 五羊运营项目名称 → 项目名称
- 承租合同/代理协议 → 租赁合同编号
- 协议开始日期 → 接收协议开始日期
- 协议结束日期 → 接收协议结束日期
- 现终端出租合同 → 终端合同文件

## 导入流程

1. 下载标准模板
2. 按规范填写数据
3. 上传Excel文件
4. 系统自动验证数据
5. 导入成功后可查看结果

## 注意事项

1. 必填字段必须填写，否则会导致导入失败
2. 日期格式请严格按照规范填写
3. 枚举字段请从指定值中选择
4. 文件名请不要包含特殊字符
5. 建议先导入少量测试数据验证无误后再批量导入

## 错误处理

系统会检查以下内容：
- 必填字段是否为空
- 数值字段是否为负数
- 枚举字段值是否有效
- 日期格式是否正确

如有错误，系统会返回详细的错误信息，请根据提示修改后重新导入。
