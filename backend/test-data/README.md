# 测试数据目录

本目录包含后端项目的测试数据库文件和测试样本数据。

## 目录结构

```
test-data/
├── databases/        # 测试数据库文件
│   ├── test_*.db    # 各种测试数据库
│   └── *.db         # 其他测试数据库
└── samples/         # 测试样本数据
    └── [待添加]
```

## 数据库文件说明

### 测试数据库
- `test_*.db` - 各种功能测试数据库
- 包含测试用的模拟数据
- 用于单元测试和集成测试

### 使用说明
1. 这些数据库仅用于测试目的
2. 不要在生产环境中使用
3. 可以根据测试需求重置或替换
4. 建议定期清理过期的测试数据

## 注意事项
- 测试数据库包含模拟数据，不包含真实用户信息
- 数据库文件可能较大，请注意磁盘空间
- 定期备份重要的测试数据配置

注意：这些文件仅用于开发和测试目的。
